## 🎯 Solution

An **AI-powered Interview Quiz Generator**  
that focuses on **high-impact topics (80/20 rule)**:

- DSA  
- OS  
- DBMS  
- CN  
- OOPs  
- Aptitude

---

## ⚙️ Features

- 🧠 AI-generated **interview-level questions**
- 🎯 Role-based quiz (Frontend / Backend / Core)
- 📊 Difficulty control (Easy / Medium / Hard)
- 📌 Topic selection (DSA, OS, etc.)
- ✅ Answers + crisp explanations
- ⚡ Fast & minimal UI (no distraction)

---

## 🧪 How It Works

1. Select:
   - Role
   - Difficulty
   - Topic  

2. Click **Start Quiz**

3. AI generates:
   - Questions  
   - Options  
   - Correct Answer  
   - Explanation  

👉 Simple flow = Better focus

---

## 🧠 Why This Works (Psychology)

- Instant feedback loop → faster learning  
- Active recall → better retention  
- Mistake-based learning → deeper understanding  

Yahi wo pattern hai jo toppers follow karte hain.
